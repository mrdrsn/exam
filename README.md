# Система мониторинга состояния здоровья пациента

## Описание проекта

Проект представляет собой синтетическую систему мониторинга состояния здоровья пациента.  
Система ведет учет пациентов, генерирует и отслеживает показатели температуры, сердечного ритма и центрального венозного давления.

Проект выполнен как экзаменационная работа по дисциплине «Теория и технология программирования».

## Цель проекта

Разработать desktop-приложение для мониторинга медицинских показателей пациента с возможностью создания пациента, запуска мониторинга, отображения текущих и предыдущих измерений, фиксации критических состояний и расчета статистических показателей.

## Выполненные задачи

В рамках проекта были выполнены следующие задачи:

- анализ технического задания;
- описание предметной области;
- формализация функциональных требований;
- построение ER-диаграммы;
- построение Use Case диаграммы;
- построение UML-диаграммы классов;
- проектирование wireframe-эскизов интерфейса;
- разработка Java-прототипа приложения;
- реализация генерации медицинских показателей;
- реализация хранения данных пациентов в лог-файлах.

## Основной функционал

- создание нового пациента;
- открытие существующего пациента из списка;
- запуск и остановка мониторинга;
- генерация температуры, сердечного ритма и центрального венозного давления;
- отображение последних 10 измерений;
- подсветка критических значений;
- фиксация времени начала критического состояния;
- фиксация времени восстановления пациента;
- расчет статистических показателей;
- хранение данных пациента в отдельном лог-файле.
